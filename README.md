# ACITY CONNECT (Frontend)

## 📌 Project Overview
ACITY Connect is a Smart Campus Marketplace and Skill Exchange platform.  
It allows students to buy and sell items, offer services, and connect with others on campus.

## 🚀 Features
- Marketplace listings
- Search and filter functionality
- Create new listings
- Login and registration interface
- Admin dashboard (UI)
